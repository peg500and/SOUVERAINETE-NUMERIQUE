# UGAIA - Dashboard Souveraineté Numérique

Dashboard React pour le suivi de la souveraineté numérique avec visualisation de données en temps réel.

## Fonctionnalités

- **Indicateurs clés (KPI)** : Taux de protection, localisation des données, incidents, conformité
- **Graphiques interactifs** :
  - Évolution des menaces et protection (Area Chart)
  - Score de souveraineté (Radar Chart)
  - Localisation des données (Pie Chart)
  - Conformité réglementaire (Bar Chart)
  - Trafic réseau 24h (Line Chart)

## Technologies

- React 19
- Vite
- Recharts

## Installation

```bash
npm install
npm run dev
```
